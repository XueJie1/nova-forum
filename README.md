# Nova Forum (新星论坛)

一个基于 Spring Boot + Redis + Elasticsearch 构建的现代化社区论坛项目。

## 项目介绍

本项目旨在实现一个功能完整、技术主流的社区平台。

## 核心技术栈

- **后端**: Spring Boot
- **数据库**: MySQL
- **数据层**: MyBatis-Plus
- **缓存**: Redis
- **搜索**: Elasticsearch
- **认证**: JWT
- **部署**: Docker
