# Nova Forum 项目基础文档

## 项目概述

Nova Forum（新星论坛）是一个基于 Spring Boot 构建的现代化社区论坛平台，致力于提供功能完整、技术主流的社区交流体验。项目采用当前最流行的技术栈，确保高性能、高安全性和良好的可维护性。

## 核心目标

- **技术先进性**: 采用 Spring Boot 3.5.7 等主流技术栈
- **功能完整性**: 提供完整的社区论坛功能（用户管理、帖子、评论、搜索等）
- **高性能架构**: 集成 Redis 缓存和 Elasticsearch 搜索优化
- **安全可靠**: 实施 JWT 认证和 Spring Security 安全防护
- **可扩展性**: 模块化设计，便于功能扩展和维护

## 项目范围

### 已完成模块
- **用户认证模块**: 注册、登录、JWT令牌管理
- **帖子模块**: 帖子发布、编辑、删除、列表查询

### 规划中的模块
- **评论系统**: 层级评论、回复功能
- **点赞功能**: Redis缓存的点赞系统
- **搜索功能**: Elasticsearch全文检索
- **Docker化部署**: 容器化部署方案

## 技术栈选择

- **后端框架**: Spring Boot 3.5.7
- **Java版本**: 17 (现代化语言特性支持)
- **数据库**: MySQL (关系型数据库) + MyBatis Plus (增强ORM)
- **缓存层**: Redis (高性能缓存)
- **搜索引擎**: Elasticsearch (全文检索)
- **安全框架**: Spring Security + JWT
- **构建工具**: Maven
- **验证框架**: Jakarta Validation

## 项目约束

- **性能要求**: 支持高并发用户访问，响应时间<200ms
- **安全要求**: 用户密码加密存储，API接口权限控制
- **兼容性**: Java 17+环境运行
- **可维护性**: 代码规范统一，注释完整，文档齐全

## 项目特色

1. **现代化技术栈**: 使用最新的 Spring Boot 版本和 Java 17
2. **安全认证机制**: JWT 无状态认证，支持分布式部署
3. **高性能缓存**: Redis 缓存热点数据，提升响应速度
4. **全文搜索**: Elasticsearch 提供智能搜索体验
5. **模块化架构**: 清晰的 MVC 分层，职责分离明确
